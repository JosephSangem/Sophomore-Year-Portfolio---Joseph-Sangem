Financial Expenses Calculator

This project is made using Python in CodeHS. This is a Financial Expenses Calculator which asks the user for each expense and stores that in a list. It then takes the total, and calculates the average and shows a message showing whether the user's monthly total spending is better or worse than tht national average. Then at the end, it prints a reciept-like-style message with each expense and how much was spent for that in a month. It also prints the total at the end.
