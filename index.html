My CodeHS Projects

Financial Expenses Calculator

Developed in Python, this project allows users to enter monthly expenses and stores them in a list. The program calculates total spending, average spending, compares the results to a national average, and prints a receipt-style summary showing all expenses and the total monthly cost.

Auto Expenses Calculator

Created in Python, this application calculates monthly vehicle expenses. The program gathers expense data from the user, calculates the total cost, and uses functions to display a detailed breakdown of expenses.

Pizza Shop Calculator

Built in C++, this project simulates a pizza ordering system. Users select pizza size and toppings, while functions calculate pricing, taxes, and generate a receipt-style bill.

Computer Shop Calculator

Developed in Python, this program simulates a computer repair shop. Users enter repair details and technician information, while functions generate a professional receipt-style summary using formatted output.
