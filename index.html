My CodeHS Projects

Financial Expenses Calculator

Developed in Python, this project allows users to enter monthly expenses and stores them in a list. The program calculates total spending, average spending, compares the results to a national average, and prints a receipt-style summary showing all expenses and the total monthly cost.

Technical Stack
  • Language: Python
  • Platform: CodeHS Sandbox
  • Concepts: Lists, Loops, User Input, Variables, Arithmetic Operations
Process & Challenges

I started by designing a system that could store multiple user expenses and calculate total and average spending. I used Python lists to store the values and loops to process them efficiently. One challenge I faced was ensuring that all user inputs were correctly added and handled without errors. I resolved this by testing different input cases and refining my logic for summing and averaging values. This project improved my understanding of data storage and list manipulation in Python.
